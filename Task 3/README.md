# Task 3

|Категории данных, которые необходимо защищать|В чем необходимость защиты|Инструменты|
|---|---|---|
|Персональные данные|Утечка данных может привести к ущербу репутации и юридическим последствиям|**Хранение**: шифрование (AES, RSA). **Передача**: SSL/TLS. **Обработка**: контроль доступа|
|Медицинская информация|Нарушение конфиденциальности может привести к серьёзным последствиям для пациентов|**Хранение**: шифрование (AES, RSA). **Передача**: SSL/TLS, Цифровые подписи и контроль целостности. **Обработка**: контроль доступа|
|Финансовая информация|Потенциальные финансовые потери и штрафы от регуляторов|**Хранение**: шифрование (AES, RSA). **Передача**: SSL/TLS, Цифровые подписи и контроль целостности. **Обработка**: контроль доступа|
|Аналитическая информация|Утечка данных может привести к ущербу репутации и юридическим последствиям|**Хранение**: обфускация. **Передача**: SSL/TLS. **Обработка**: анонимизация|

## Механизм тегирования данных

### Набор тегов

|Тег|Описание|
|---|---|
|`PII`|персональная информация|
|`PHI`|медицинская информация|
|`Клинические данные`|Анализы|

### Правила назначения

Теги назначаются автоматически, в зависимости от источника данных. Регистрация пользователя - тег `PII`, врач вводит диагноз - `PHI`, получены результаты анализов из лаборатории - `Клинические данные`

## Реестр программных и аппаратных средств для защиты данных

|Наименование|Состояние данных|Описание|
|---|---|---|
|Veeam Backup & Replication|Хранение|Инструмент для архивации и резервного копирования с шифрованием. Может использоваться для хранения устаревшей аналитической информации и истории данных о пациенте|
|OpenSSL|Передача|Инструмент для реализации протоколов SSL/TLS|
|OpenVPN|Передача|Инструмент для реализации виртуальных приватных сетей. Может использоваться для удаленного администрирования приложения или, например, для удаленного подключения врачей. |
|BitLocker|Хранения|Инструмент для шифрования дисков на Windows-системах.|
|TIBCO Spotfire|Обработка|Средство для визуализации данных с возможностью соблюдения политик безопасности и управления доступом|
|Keycloak|Хранение, передача, обработка|IAM для контроля доступа|
